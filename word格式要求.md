# 复旦大学计算与智能创新学院本科毕业论文格式规范整理

> 来源：`复旦大学计算与智能创新学院本科毕业论文撰写规范(3).docx`
> 用途：将论文撰写规范中与“论文材料、结构、排版、图表公式、份量、审查、AI 工具使用”有关的要求抽取为 Markdown 检查清单，便于对照 Word / LaTeX 模板。

---

## 1. 毕业论文存档材料组成

毕业论文存档材料包括：

* 《毕业论文开题报告》
* 《毕业论文教师指导记录表》
* 《毕业论文中期检查记录》
* 《毕业论文相重性检测报告》
* 《毕业论文正文》

  * 封皮
  * 目录
  * 中文摘要
  * 英文摘要
  * 正文
  * 参考文献
  * 致谢

如相关材料需要手写：

* 应按要求认真填写
* 字迹工整
* 卷面整洁
* 一律使用碳素或蓝黑墨水钢笔填写

---

## 2. 毕业论文正文装订顺序

毕业论文正文应按统一标准装订，顺序为：

```text
封面 → 目录 → 中英文摘要 → 正文 → 参考文献 → 致谢
```

---

## 3. 论文标题格式与内容要求

### 3.1 标题排版

| 项目 | 要求 |
| -- | -- |
| 字体 | 黑体 |
| 字重 | 加粗 |
| 字号 | 小二 |
| 对齐 | 居中 |

### 3.2 标题内容

标题应满足：

* 简短
* 明确
* 有概括性
* 能让读者大致了解：

  * 课题内容
  * 专业相关理论
  * 方法重点
  * 技术重点
  * 论文亮点

标题字数要求：

* 一般不宜超过 20 字
* 若细节较多，为避免标题冗长，可以拆分为：

  * 主标题：简明概括
  * 副标题：补充细节

---

## 4. 目录格式要求

### 4.1 “目录”二字格式

| 项目 | 要求 |
| -- | -- |
| 字体 | 黑体 |
| 字号 | 二号 |
| 对齐 | 居中 |

### 4.2 目录正文格式

| 项目     | 要求           |
| ------ | ------------ |
| 目录标题下方 | 下空一行         |
| 内容     | 章、节、小节及其开始页码 |
| 字号     | 小四           |
| 行距     | 20 磅         |
| 章名     | 黑体、加粗        |
| 其他条目   | 宋体           |

### 4.3 目录层级要求

目录按三级标题编写：

```text
第一章 ……
1.1 ……
1.1.1 ……
```

要求：

* 标题层次清晰
* 目录中的标题必须与正文中的标题一致

## 5. 中文摘要与关键词格式要求

### 5.1 “摘要”二字格式

| 项目 | 要求 |
| -- | -- |
| 字号 | 小二 |
| 字体 | 黑体 |
| 对齐 | 居中 |

### 5.2 摘要正文格式

| 项目 | 要求      |
| -- | ------- |
| 字号 | 小四      |
| 字体 | 宋体      |
| 行距 | 20 磅    |
| 段落 | 首行缩进二个字 |

### 5.3 关键词格式

摘要内容后空一行打印“关键词”。

| 项目      | 要求            |
| ------- | ------------- |
| “关键词”三字 | 小三号黑体         |
| 关键词内容   | 小四号宋体         |
| 关键词数量   | 5 个           |
| 分隔方式    | 各关键词之间用逗号分开   |
| 末尾标点    | 最后一个关键词后无标点符号 |

## 6. 英文摘要与英文关键词格式要求

英文摘要及关键词要求：

* 字体及书写格式最好与中文摘要对应
* 英文题目一律使用大写字母
* 英文关键词应与中文关键词对应
* 中英文关键词顺序应一致

---

## 7. 摘要内容要求

摘要应包括以下内容：

1. 课题研究的学科背景、目的以及意义
2. 研究的主要内容，以及采用的核心技术和方案等
3. 主要成果及其价值
4. 简要的研究结论和效果总结

摘要写作要求：

* 准确、凝练地概括论文主要内容
* 一般不做主观评价
* 文字精炼、清晰、准确、朴素
* 一般不举例证
* 不详细阐述实现过程
* 不使用图表
* 使用标准专业术语、惯用缩写和符号
* 字数一般不超过正文的 5%

---

## 8. 正文格式要求

### 8.1 正文整体

正文可分为若干章。

| 项目   | 要求       |
| ---- | -------- |
| 每章标题 | 小二号黑体，居中 |
| 正文内容 | 小四号宋体    |
| 字符间距 | 标准       |
| 行距   | 20 磅     |
| 段落   | 首行缩进二个字  |

### 8.2 公式、表格、插图编号

| 对象   | 要求               |
| ---- | ---------------- |
| 公式   | 另起一行，公式序号按章节顺序编号 |
| 表格   | 按章节顺序编号          |
| 插图   | 按章节顺序编号          |
| 图中坐标 | 应标注单位            |

---

## 9. 引言内容要求

引言作为论文主体内容的引入，位于正文之前。

引言通常写 1 章，主要说明论文主题和纲要，通常包括：

1. 课题提出背景、技术沿革等
2. 课题研究目的和意义
3. 本文核心内容和创新点
4. 论文结构

---

## 10. 文献综述与前导知识要求

国内外文献资料的分析与综述，以及论文必需的前导知识简介，通常写 1 到 2 章。

该部分应承担：

* 梳理国内外相关研究
* 分析已有工作的特点与不足
* 引出本文研究问题
* 介绍理解论文主体所需的前导知识

---

## 11. 论文主体部分要求

论文主体部分通常写 2 章，可包括：

* 系统分析与总体设计
* 算法分析和模型建立
* 详细设计与实现
* 实验验证和效果分析

主体部分是论文核心，应做到：

* 系统设计与实现说明充分、详细
* 算法设计与推理详尽
* 条理清晰
* 逻辑严密
* 阐述充分
* 形成具有应用创新的系统构建，或在特定领域提出优于已有方法的算法设计和改进

---

## 12. 系统分析、设计与实现要求

主体工作一般包括以下内容：

### 12.1 系统的分析与总体设计

需要说明：

* 系统需求
* 总体架构
* 模块划分
* 技术路线
* 设计依据

### 12.2 重点突出的详细设计与实现

需要：

* 叙述重点和关键内容
* 突出本人工作的创新点
* 强调研究过程和成果中的科学规律
* 如引用他人工作，应标出参考文献序号
* 不必详述他人工作，重点应放在本人研究内容上
* 如对他人方法有改进，应进行对比分析

若论文属于创新型应用系统构建：

* 应按照软件工程过程进行阐述
* 要求条理清晰、重点突出

若论文属于专业理论或方法创新：

* 应按照科学论证规律进行阐述
* 应说明改进理论或方法的适用范围
* 应说明局限性
* 力求论证具有科学性和逻辑性

---

## 13. 研究结果与分析要求

研究结果与分析是整篇论文的核心部分。

该部分需要：

* 对研究结果作详实、具体的判断分析
* 尽量避免阐述众所周知的常识性内容
* 突出论文主要工作

一般应包括：

1. 验证相关的主要专业原理或概念
2. 说明实验条件

   * 如研究在一定适用范围或限制性条件下开展，应突出说明
3. 将课题研究结果与他人研究工作进行对比分析

   * 对比分析应作为重点
   * 应突出本人工作
4. 解释其中的科学规律

   * 如因果关系
   * 论证其必然性或偶然性
5. 提出本研究没有解决的、或尚需进一步探索的问题

写作时应围绕：

* 问题之间的相关性
* 因果关系
* 异常情况
* 逻辑严谨性

---

## 14. 结论格式与内容要求

### 14.1 结论格式

结论通常为最后一章。

| 项目   | 要求       |
| ---- | -------- |
| 标题   | 小二号黑体，居中 |
| 内容   | 小四号宋体    |
| 字符间距 | 标准       |
| 行距   | 20 磅     |
| 段落   | 首行缩进二个字  |

### 14.2 结论内容

结论是整个课题研究的总结，应：

* 对研究全过程、实验结果等进行综合分析
* 准确反映客观事物的本质及规律
* 与引言相呼应
* 与正文其他部分相联系
* 具有说服力
* 语言准确、鲜明
* 观点肯定或否定都应有事实根据

结论不应：

* 简单复述前文论述结果
* 想当然
* 含糊其词
* 使用“大概”“可能”“或许”等不确定表述

---

## 15. 参考文献格式与引用要求

### 15.1 参考文献排版

| 项目   | 要求                  |
| ---- | ------------------- |
| 编号方式 | 按论文中出现次序，用中括号数字连续编号 |
| 内容字体 | 小四号宋体               |
| 字符间距 | 标准                  |
| 行距   | 20 磅                |
| 外文资料 | 保持原文，不得涂写           |

### 15.2 参考文献信息

参考文献应依次书写：

* 作者
* 文献名
* 杂志或书名
* 卷号或期刊号
* 出版时间

### 15.3 文中引用要求

论文中凡引用他人的：

* 报告
* 论文
* 观点
* 数据
* 材料
* 成果

都应：

* 按引用先后顺序排列
* 在文中标明参考文献顺序号或引文作者姓名
* 在参考文献列表中按规定格式列出

注意：

* 列入参考文献的文献，作者应详细阅读过
* 不应列入未曾阅读的文献

---

## 16. 致谢要求

致谢应放在论文最后。

内容要求：

* 用简短文字表达谢意
* 感谢论文研究与写作过程中给予帮助和支持的单位和个人
* 可包括指导教师等

写作原则：

* 简洁
* 礼貌
* 尊重他人劳动
* 体现治学态度

---

## 17. 文稿与语言要求

毕业论文应：

* 一律采用国家文字改革委员会正式公布的简化汉字书写
* 一律采用计算机排版、打印
* 文字通顺
* 语言流畅
* 版面整洁

论文相关材料如需填写：

* 必须用碳素或蓝黑墨水钢笔书写

---

## 18. 封皮要求

论文采用复旦大学本科毕业论文（设计）统一封皮。

---

## 19. 页眉与页脚要求

### 19.1 页眉

毕业论文主体部分应标注页眉。

| 位置   | 内容                      |
| ---- | ----------------------- |
| 页眉左侧 | 论文主标题                   |
| 页眉右侧 | 相应章节和章节标题，例如“第一章 XXXXX” |

### 19.2 页脚与页码

| 部分   | 页码格式              | 位置 |
| ---- | ----------------- | -- |
| 目录   | 罗马数字，如 I、II、III   | 居中 |
| 从引言起 | 阿拉伯数字，如 1、2、3、4…… | 居中 |

---

## 20. 名词、名称与外文缩写要求

### 20.1 科学技术名词

科学技术名词术语应尽量采用：

* 全国自然科学名词审定委员会公布的规范词
* 国家标准规定的名称

对于尚未统一规定或叫法有争议的名词术语：

* 可采用惯用名称

### 20.2 外文缩写

使用外文缩写代替某一名词术语时：

* 首次出现时应在括号内注明其含义

示例：

```text
CPU（Central Processing Unit）
```

### 20.3 外国人名

外国人名一般采用英文原名，可不译成中文。

英文人名书写原则：

* 按名前姓后的原则书写
* 不可省略名的部分

示例：

```text
P. Cray
```

不应写成：

```text
Cray
```

一般熟知的外国人名，如牛顿、爱因斯坦、达尔文、马克思等，可按通常标准译法写译名。

---

## 21. 量和单位要求

毕业论文中的量和单位必须采用：

```text
中华人民共和国国家标准 GB3102-93
```

该标准以国际单位制为基础。

非物理单位，如件、元等，可用汉字与符号构成组合形式的单位。

示例：

```text
件/台
元/kg
```

---

## 22. 数字使用要求

毕业论文中的测量、统计数据一律使用阿拉伯数字。

示例：

```text
5.25kg
```

---

## 23. 公式要求

所有公式不得徒手书写。

公式应使用以下工具之一编辑：

* Microsoft 公式编辑器
* LaTeX
* MathType
* 其他公式编辑软件或工具

正文中的公式要求：

* 另起一行
* 按章节顺序编号

---

## 24. 表格要求

每个表格应包含：

* 表序
* 表题

表格格式要求：

* 表题写在表格上方正中
* 表序写在表题左方
* 表序后不加标点
* 表序后空一格接写表题
* 表题末尾不加标点

编号要求：

* 全文表格可以统一编序
* 也可以逐章编序
* 无论采用哪种方式，表序必须连续

跨页表格要求：

* 如表格需从下页接写，接写时省略表题
* 表头应重复书写
* 右上方写“续表 XX”

位置要求：

* 表格应位于其支撑的相关正文阐述附近
* 不应过分超前或拖后

---

## 25. 插图要求

插图制作要求：

* 精心制作
* 线条匀称
* 图面整洁美观
* 与正文呼应
* 不得与正文脱节

每幅插图应包含：

* 图序
* 图题

编号要求：

* 全文插图可以统一编序
* 也可以逐章单独编序
* 无论采用哪种方式，图序必须连续
* 不得重复
* 不得跳缺

若插图由若干分图组成：

* 分图用 `a, b, c, ...` 标序
* 分图图名以及图中各种代号的意义，以图注形式写在图题下方
* 图注先写分图名
* 另起一行后写代号意义

---

## 26. 论文字数与外文文献阅读量要求

毕业论文字数要求：

* 一般不少于 1.5 万字
* 或达到相当信息量

外文文献阅读量：

* 由指导教师量化具体要求

---

## 27. 毕业论文规范审查要求

论文规范审核工作：

* 由指导教师具体负责
* 从论文的质、量、形式等规范方面审查答辩资格

审查结果：

* 审查合格者方能参加答辩
* 质、量、形式等方面审查不合格者，应责令返工
* 返工直到达到要求为止

校外进行毕业论文的学生：

* 论文答辩资格由各联系导师负责审查

---

## 28. AI 工具使用规范

毕业论文写作过程中，如需使用人工智能工具：

* 应事先征得指导教师同意
* 必须严格遵照《复旦大学关于在本科毕业论文（设计）中使用AI工具的规定（试行）》（2024 年 9 月）
* 必须按上述规定第五条进行披露与声明
* 必须保证论文导师对作者使用人工智能工具的情况知情并同意

---

## 29. 推荐论文结构模板

根据规范，可以整理出如下本科毕业论文结构：

```text
封面
目录
中文摘要
中文关键词
英文摘要
英文关键词

第一章 引言
  1.1 研究背景
  1.2 研究目的与意义
  1.3 本文核心内容与创新点
  1.4 论文结构

第二章 国内外研究现状与前导知识
  2.1 国内外研究现状
  2.2 相关理论与技术基础
  2.3 已有方法的问题与本文切入点

第三章 系统分析与总体设计 / 方法设计
  3.1 问题定义
  3.2 总体框架
  3.3 关键模块或算法设计

第四章 详细设计与实现 / 实验设计
  4.1 实现细节
  4.2 实验设置
  4.3 数据、环境与评价指标

第五章 实验结果与分析
  5.1 主要实验结果
  5.2 对比实验
  5.3 消融实验
  5.4 结果分析与讨论
  5.5 局限性与进一步工作

第六章 结论

参考文献
致谢
```

* [ ] 标题为黑体、加粗、小二、居中

* [ ] 目录二字为黑体、二号、居中
* [ ] 目录正文为小四号，行距 20 磅
* [ ] 目录按三级标题编写
* [ ] 目录标题与正文标题一致

### 30.3 摘要与关键词

* [ ] 中文摘要标题为小二号黑体居中
* [ ] 摘要正文为小四号宋体，行距 20 磅，首行缩进二字
* [ ] 关键词为 5 个
* [ ] 关键词之间用逗号分隔
* [ ] 最后一个关键词后无标点
* [ ] 英文题目全部大写
* [ ] 中英文关键词顺序对应

### 30.4 正文与结论

* [ ] 每章标题为小二号黑体居中
